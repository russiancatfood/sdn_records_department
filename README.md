# SDN Records Department #

A simple utility for calculating your running game stats for Dispatch (and AdHoc Studios game). The tool runs entirely on the front end, so it's fast and requires no back-end infrastructure.

Drop your SaveSlot file into the floppy drive 💾 (ask your parents) and see where you stand on important game metrics as you progress through the game episodes.

Note:

This is a fan-made utility and I'm in no way associated with AdHoc Studios.
